1 jQuery处理事件示例

1.1 问题

1）动态的给按钮绑定单击事件

2）单击时输出鼠标点击的坐标

1.2 方案

绑定单击事件：$(选择器).click(function(e){});

获取事件源坐标：e.pageX，e.pageY

1.3 步骤

步骤一：创建HTML

创建demo1.html，代码如下：

copytextpop-up
  1. <!DOCTYPE html>
  2. <html>
  3.   <head>
  4.     <title>jQuery事件</title>
  5.     <meta charset="utf-8" />
  6.     <script type="text/javascript" src="../js/jquery-1.11.1.js"></script>
  7.     <script type="text/javascript">
  8.       $(function(){
  9.         $(":button").click(function(e){
  10.           alert(1);
  11.           alert(e.pageX + " : " + e.pageY);
  12.         });
  13.       });
  14.     </script>
  15.   </head>
  16.   <body>
  17.     <input type="button" value="按钮一"/>
  18.   </body>
  19. </html>

<html>

3 合成事件示例

3.1 问题

1）增加一张图片

2）鼠标悬停在图片上时让图片变大

3）鼠标移出时让图片变回原始大小

3.2 方案

hover(mouseenter，mouseleave) 模拟光标悬停事件

3.3 步骤

步骤一：创建HTML

创建demo3.html，代码如下：

copytextpop-up
  1. <!DOCTYPE html>
  2. <html>
  3.   <head>
  4.     <title>合成事件</title>
  5.     <meta charset="utf-8" />
  6.     <script type="text/javascript" src="../js/jquery-1.11.1.js"></script>
  7.     <script type="text/javascript">
  8.       $(function(){
  9.         $("img").hover(
  10.           function(){
  11.             $(this).width("300px").height("300px");
  12.           },
  13.           function(){
  14.             $(this).width("218px").height("218px");
  15.           }
  16.         );
  17.         
  18.         $(":button").click(function(){
  19.           $("img").toggle();
  20.         });
  21.       });
  22.     </script>
  23.   </head>
  24.   <body>
  25.     <p><input type="button" value="切换"/></p>
  26.     <p><img src="../images/01.jpg" /></p>
  27.   </body>
  28. </html>
<!DOCTYPE html>
<html>
  <head>
    <title>合成事件</title>
    <meta charset="utf-8" />
    <script type="text/javascript" src="../js/jquery-1.11.1.js"></script>
    <script type="text/javascript">
      $(function(){
        $("img").hover(
          function(){
            $(this).width("300px").height("300px");
          },
          function(){
            $(this).width("218px").height("218px");
          }
        );
        
        $(":button").click(function(){
          $("img").toggle();
        });
      });
    </script>
  </head>
  <body>
    <p><input type="button" value="切换"/></p>
    <p><img src="../images/01.jpg" /></p>
  </body>
</html>

4 模拟操作示例

4.1 问题

1）增加一组姓名和文本框

2）点击[自动打分]按钮时，逐个切入文本框，并随机打分

4.2 方案

模拟光标切入事件：$obj.trigger(“focus”);

4.3 步骤

步骤一：创建HTML

创建demo4.html，代码如下：

copytextpop-up
  1. <!DOCTYPE html>
  2. <html>
  3.   <head>
  4.     <title>模拟操作</title>
  5.     <meta charset="utf-8" />
  6.     <script type="text/javascript" src="../js/jquery-1.11.1.js"></script>
  7.     <script type="text/javascript">
  8.       function auto_mark() {
  9.         var texts = $(":text");
  10.         var i = 0;
  11.         var timer = setInterval(function(){
  12.           if(i == texts.length) {
  13.             clearInterval(timer);
  14.           }
  15.           var score = Math.round(Math.random()*100);
  16.           texts.eq(i++).trigger("focus").val(score);
  17.         }, 1000);
  18.       }
  19.     </script>
  20.   </head>
  21.   <body>
  22.     <p><input type="button" value="自动打分" onclick="auto_mark();"/></p>
  23.     <p>张三：<input type="text" /></p>
  24.     <p>李四：<input type="text" /></p>
  25.     <p>王五：<input type="text" /></p>
  26.     <p>马六：<input type="text" /></p>
  27.     <p>周七：<input type="text" /></p>
  28.   </body>
  29. </html>
<!DOCTYPE html>
<html>
  <head>
    <title>模拟操作</title>

    <script type="text/javascript" src="../js/jquery-1.11.1.js"></script>
    <script type="text/javascript">
      function auto_mark() {

</html>

5 jQuery动画示例

5.1 问题

1）增加一张图片

2）使用不同的方式实现该图片显示/隐藏的动画效果

3）自定义动画，使得改图片向右移动后再向下移动

5.2 方案

显示/隐藏动画效果

show() / hide()

slideDown() / slideUp()

fadeIn() / fadeout()

自定义动画效果

animate（偏移位置，执行时间，回调函数）

5.3 步骤

步骤一：创建HTML
